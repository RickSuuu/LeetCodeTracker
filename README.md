# LeetCode 热题 100 · 桌面追踪器

macOS 原生悬浮小窗口应用，帮你按类型刷 LeetCode 热题 100，追踪完成进度。

![macOS](https://img.shields.io/badge/macOS-13%2B-blue)
![Swift](https://img.shields.io/badge/Swift-5.9-orange)

## 特性

- 🔥 完整收录 LeetCode 热题 100，按 17 个算法类型分组
- ✅ 勾选完成状态，进度自动保存（`~/.leetcode-tracker/progress.json`）
- 🔍 支持搜索和筛选（全部/未完成/已完成）
- 🖱️ 点击题目名直接跳转力扣题目页
- 🪟 macOS 原生毛玻璃悬浮窗口，always-on-top
- 📊 实时进度条和完成统计
- 🫥 菜单栏驻留，不占 Dock 位置

## 构建 & 运行

```bash
# 编译 + 打包
bash bundle.sh

# 运行
open LeetCodeTracker.app
```

需要 macOS 13+ 和 Swift 5.9+。
